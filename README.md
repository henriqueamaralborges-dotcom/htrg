# Echo of Silence - Guia de Deploy

Este jogo foi criado para ser simples, performático e emocionante. Siga os passos abaixo para fazer o deploy no GitHub Pages.

## 🚀 Como fazer o Deploy

1. **Abra o GitHub Desktop**:
   - Clique em `File` > `Add Local Repository`.
   - Selecione a pasta deste projeto: `echo-of-silence`.
   - Clique em `Create a repository here` se ele ainda não for um repositório Git.

2. **Publicar no GitHub**:
   - Dê um nome ao repositório (ex: `echo-of-silence`).
   - Clique em `Publish repository`. Certifique-se de que a opção "Keep this code private" esteja **desmarcada** se quiser usar o GitHub Pages gratuito.

3. **Ativar o GitHub Pages**:
   - Vá para o seu repositório no site do GitHub.
   - Clique em **Settings** (Configurações) > **Pages**.
   - Em "Build and deployment", selecione a branch `main` e a pasta `/ (root)`.
   - Clique em `Save`.

4. **Acessar o Jogo**:
   - Após alguns minutos, seu jogo estará disponível na URL fornecida pelo GitHub (geralmente `https://seu-usuario.github.io/echo-of-silence/`).

## 🎮 Mecânicas do Jogo

- **A Experiência do Silêncio**: O jogo não possui som. Você deve "ouvir" com os olhos.
- **Siga o Echo**: Se você se afastar muito de Echo, o mundo começará a ficar borrado e escuro. Mantenha-se por perto para enxergar o caminho.
- **Movimentação**: Use o mouse para guiar Lyra pelo Limbo das Frequências.
- **Vibração**: A tela tremerá em momentos de perigo ou impacto emocional.
